Instructions for creating input text file:

1. The execution of the program needs an input text file.
2. The input file should have two arguments n and k
3. n represents the number of random points to be generatesd to calculate the value of pi.
4. k represents the number of threads used to do said task.
5. The two arguments n and k either with a space in between or in different lines.

Instructions for executing Assgn1_Src_ES21BTECH11022 :

1. First compile the c file via the command "g++ Assgn1_Src_ES21BTECH11022.cpp"/.
2. This creates the output file a.out.
3. To execute the code use the command "./a.out <name>.txt".
4. The output would be a text file.
5. The Output file will contain the following details:
    a)Time taken to calculate value of pi.
    b)Calculated value of pi.
    c)The log of each individual thread containing all the points checked and those obtained within the circle and sqaure.
